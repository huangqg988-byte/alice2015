游戏说明书

游戏简介

《谁是卧底》是一款经典的聚会推理游戏，适合4-12人一起玩。游戏中有三种角色：平民、卧底和白板。平民和卧底会拿到相似但不同的词语，白板则没有词语。通过描述和推理，找出隐藏在人群中的卧底！

角色介绍

平民 拿到"平民词"，人数最多，目标是找出卧底并投票淘汰他们
卧底 拿到"卧底词"（与平民词相似），人数较少，目标是隐藏身份存活到最后
白板 没有词语，需要通过观察和推理来判断自己可能的身份，存活即可获胜

游戏流程

  1. 设置阶段：选择玩家人数、难度级别、时间设置等
  2. 看牌阶段：按顺序传递手机，每人有几秒钟时间查看自己的身份词语
  3. 描述阶段：轮流描述自己的词语（不能直接说出词语），每轮结束后投票
  4. 投票阶段：讨论后选择要淘汰的玩家编号
  5. 遗言阶段：被淘汰者有短暂时间发表遗言
  6. 胜负判定：若卧底全部淘汰则平民胜；若卧底人数≥平民或仅剩2人则卧底胜

获胜条件

  • 平民获胜：所有卧底被投票淘汰
  • 卧底获胜：
    • 卧底人数 ≥ 平民人数
    • 投票次数用尽
    • 场上仅剩2名玩家
  • 白板获胜：存活到游戏结束（跟随所在阵营）

难度说明

简单1名卧底，适合新手
普通1名卧底 + 1名白板，推荐模式
困难2名卧底，需要更细致的观察
专家2名卧底 + 1名白板，高手挑战
自定义自由设置卧底和白板数量

游戏技巧

  • 描述要模糊但有关联：太详细容易暴露，太模糊会被怀疑
  • 观察投票行为：卧底往往会投票给平民以保护自己
  • 注意描述差异：卧底看到的词和平民不同，描述会有细微差别
  • 白板要学会"装"：根据他人描述推断可能的词语
  • 不要急着投票：多听几轮描述再做判断

快捷键

空格键翻开下一张牌 / 跳过描述
回车键确认投票
F9显示/隐藏裁判信息（主持人专用）
F10显示/隐藏游戏历史
F11显示游戏统计
ESC暂停/恢复游戏
Ctrl + S保存游戏进度

注意事项

  • 看牌时请确保周围没人偷看！
  • 传递手机时等待"传递提示"出现
  • 描述时禁止直接说出词语或使用谐音
  • 被淘汰后禁止继续发言或暗示
  • 建议使用裁判模式，由一人负责操作手机
